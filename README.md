# Math Trainer

A simple math study assistant app for learning elementary level math.

Runs as a [Proressive Web App](https://web.dev/progressive-web-apps/).

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development / watch mode
$ npm run start
```

## Test

```bash
# unit tests
$ npm run test
```

## Build

```bash
# production build
$ npm run build
```
